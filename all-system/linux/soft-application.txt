# ****************************** 其它软件 ******************************
- 其它重要软件
   1. 声卡
      - pacman -S alsa-utils           # 安装音频工具
         - alsa-mixer                  # 调起声音控制台
         - alsa 默认在内核所以只需安装调度工具
         - 包含 alsamixer, amixer 命令
         - 默认静音，解决方案
            - amixer sset Master unmute
            - amixer sset Speaker unmute
            - amixer sset Headphone unmute

   2. 蓝牙
      - pacman -S bluez                   # 提供蓝牙协议栈
      - pacman -S bluez-utils             # 提供蓝牙 bluetoothctl 命令工具
      - pacman -S blueman                 # 蓝牙可视化工具（推荐）
      - 要使用蓝牙，需要将使用用户加入 lp 的组中
         - usermod -G lp 用户名
      - pacman -S pulseaudio-bluetooth    # 以支持蓝牙麦克风和蓝牙话筒等特性
         - pulseaudio -k                  # 重启 pulseaudio 使安装生效
      - 将所有音频输出到蓝牙设备
         - 编辑 /etc/pulse/system.pa 添加以下两行
         ```
         load-module module-bluetooth-policy
         load-module module-bluetooth-discover
         ```
      - 注意
         - 可能因节能等原因 造成蓝牙键盘无法连接或其它因为节能造成的问题
         - 熟悉后再特殊配置以支持蓝牙键盘连接，暂时不安装，安装需要特殊配置

   3. Linux 视频剪辑软件
      - https://www.linuxprobe.com/linux-best-video.html
      - 如果你需要一个简单的视频编辑器，OpenShot、Kdenlive 和 Flowblade 是一个不错的选择。这些软件是适合初学者的，并且带有标准规范的系统。
      - 如果你有一个高性能的计算机，并且需要高级功能，你可以使用 Lightworks。如果你正在寻找更高级的功能， Blender 可以帮助你。
      - 这就是我写的 5 个最佳的视频编辑软件，它们可以在 Ubuntu、Linux Mint、Elementary 和其他 Linux 发行版下使用。 请与我们分享您最喜欢的视频编辑器。
      - 注意
         - lightworks 有两个版本：Lightworks 免费版和 Lightworks 专业版。
         - 不过免费版本不支持 Vimeo（H.264 / MPEG-4）和 YouTube（H.264 / MPEG-4） - 高达 2160p（4K UHD）、蓝光和 H.264 / MP4 导出选项，
         - 以及可配置的位速率设置，但是专业版本支持
      - 软件
         - Kdenlive   通用视频编辑       自由开源
         - OpenShot   通用视频编辑       自由开源
         - Shotcut    通用视频编辑       自由开源
         - Flowblade  通用视频编辑       自由开源
         - Lightworks 专业级视频编辑     免费增值
         - Blender    专业级三维编辑     自由开源
         - Cinelerra  通用视频编辑       自由开源
         - DaVinci    专业级视频处理编辑 免费增值
         - VidCutter  简单视频拆分合并   自由开源
