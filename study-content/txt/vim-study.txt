* 关于 cscope 的问题
  - https://www.jianshu.com/p/3233d829697a
  - 命令
     - cscope -Rbqk
        - -R: 在生成索引文件时，搜索子目录树中的代码
        - -b: 只生成索引文件，不进入cscope的界面
        - -k: 在生成索引文件时，不搜索/usr/include目录
        - -q: 生成cscope.in.out和cscope.po.out文件，加快cscope的索引速度
  - 讲关于 cscope 的问题：
     - https://www.jianshu.com/p/3233d829697a

* 一些命令
   - :set ignorecase # 排除大小写
   - :set noignorecase # 取消排除大小写
   - . 对你作出的改变进行重复（除了 u, ctrl-r，: 命令行的改变）
   - cis/cas # 删除一个句子，并进入插入模式
      - 句子末的空白不删除，cas 会删除句末空白
   - gu/gU # 转换为小写/大写
   - ~ # 交换大小写

* 翻页后光标的位置
   - zz # 翻半页，但是光标在屏幕中间
   - zt # 翻半页，但是光标在顶部
   - zb # 翻半页，但是光标在底部

* 分屏
   - 分屏后的操作
      - ctrl = # 屏幕占屏比例相同
      - ctrl | # 当前窗口占全屏（对于垂直切分的情况）
      - ctrl _ # 屏幕占屏比例相同（对于水平切分的情况）
   - 其它
      - 如果打开了多个窗口，:only 关闭其它的所有窗口
      - 改变窗口高
         - [number]ctrl+w +/- # 增加/减少窗口的高（number 是数字）
      - 窗口光标 移动
         - ctrl+w h/j/k/l 或 H/J/K/L # 小写则是光标移动，大写则是窗口移动

* 注意
   - 以下基本出现的 n 是一个数字
      - n ; 重复 最后一次使用 f,F,t,T 的命令
      - n , 重复 最后一次使用 f,F,t,T 的命令，反方向
      - n - 向上，移到第一个非空位置
      - n + 向下，移到第一个非空位置
      - n ) 正向偏移几个句子
      - n ( 反向偏移几个句子
      - n } 正向偏移几个段落
      - n { 反向偏移几个段落

* 记号跳转
   - 跳转
      - 注意
         - ` 是跳转
         - ' 也是跳转，但是是在位置所在行的第一个非空位置
      - ctrl i # 跳转到较新的位置标记
      - ctrl o # 跳转到较旧的位置标记
      - ctrl t # 退出标记栈
      - ctrl ] # 进入标记
   - 命令
      - :marks 查看活跃的标记
      - :ju 打印跳转的清单
      - m[a-zA-Z] 为当前位置作一个标记（小写为在本文件中的，大写为跨文件的全局）
      - `[a-z] 在当前文件中，跳转到 相应的位置
      - `[A-Z] 跳转到任何文件的[A-Z]位置
      - `[0-9] 跳转到 vim 最近退出的位置
      - `` 跳转到之前跳转的位置
      - `" 跳转到最近编辑的位置
      - `[ 跳转到最近操作或输出文本的位置的开头
      - `] 跳转到最近操作或输出文本的位置的结尾
      - `< 跳转到最近 visual area 的开头
      - `> 跳转到最近 visual area 的结尾
      - `. 跳转到这个文件最后改变的位置

* buffers
   - vim file1 file2 file3 ... # 编辑多个文件
      - 文件的选择
         - :edit file # 放弃当前文件编辑，编辑新的文件，如果没有保存更改会弹出错误
         - :edit! file # 放弃当前文件编辑的更改，转而编辑新的文件
         - :hide edit file # 隐藏当前文件，转而编辑新文件
         - args # 查看多个编辑文件的清单，当前编辑文件在 [] 中
         - first # 到第一个文件
         - last # 到最后一个文件
         - :next # 编辑下一个文件，当前文件暂时在缓冲区中
         - :2next # 编辑下一个之后的文件
         - :next! # 编辑下一个文件，当前文件暂时在缓冲区中，放弃当前文件的更改
         - :wnext # 编辑下一个文件，保存当前文件的更改
         - :previous # 同 next 差不多，是编辑前一个文件（修改情况也同理）
         - :2previous # 编辑上一个之前的一个文件
      - 变更编辑清单
         - args file4 file5 file6
         - ctrl+^ # 在最近编辑的两个文件之间切换
         - `" # 在跳转到一个文件时，按这个光标会移动到你离开这个文件时的位置
         - `. # 在跳转到一个文件时，按这个光标会移动到你最后修改这个文件的位置
