* 插件安装
   - 其它插件
      - Git History # Git 提交记录界面
      - Git Graph   # Git 分支流
      - vim
         - 注意设置 Vim:Incsearch 勾选上
      - CodeGeeX    # 代码生成器
   - C/C++
      - 安装基础环境
         - 安装 gcc（windows 应该安装 MinGW 并配置环境变量）
            - MinGw下载地址 https://sourceforge.net/projects/mingw-w64/files/mingw-w64/mingw-w64-release/
            - 执行 g++ -v 命令检测是否环境变量配置成功
      - 安装插件
         - C/C++
         - C/C++ Extension Pack
         - C/C++ Themes
         - Code Runner
         - gdb    # 这是debug调试工具，否则debug调试时会出错
   - JAVA
      - java 支持插件
         - Extension Pack for Java
         - Maven for Java
            - 需要配置 mvn 的执行文件路径
            - 这里配置 settings.xml 文件路径不生效
         - Debugger for Java 装哪个？
         - Java Debugger 装哪个？
         - Language Support for Java(TM) by Red Hat
            - 需要在配置的 Java>Configuration>Maven:Global Settings 配置 maven 的 settings.xml 文件路径
   - Spring
      - Spring Initializr Java Support
      - Spring Boot Tools
      - Spring Boot Dashboard
      - Spring Boot Extension Pack
   
* 运行需知
   - java
      - Debug 配置 launch.json，具体配置可根据 ctrl + space 提示，也可根据官网介绍配置
      - 一般注重的两个配置
         - 命令参数 "args": ["--spring.profiles.active=FAT"]
         - 环境变量 "env": {"env": "FAT"]
