* 复杂度
   - 说明
      - 复杂度通常用 O(n) 表示，但是注意 n 是数据规模，而不一定是参数
   - 类别
      - 最好情况复杂度
      - 最坏情况复杂度
      - 平均情况复杂度
         - (从最好的情况复杂度O(1) + O(2) + ... + 到最坏的复杂度 O(n) ) / n
      - 均滩复杂度
      - 复杂度震荡
